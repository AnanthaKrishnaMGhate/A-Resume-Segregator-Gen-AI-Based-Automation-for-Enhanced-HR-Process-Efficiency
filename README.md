# A-Resume-Segregator-Gen-AI-Based-Automation-for-Enhanced-HR-Process-Efficiency
This work presents Resume Segregator, a Generative AI–based HR automation framework that uses Groq’s LLaMA-3, Python, PyPDF2, and Streamlit to analyze resumes, score candidates, classify them as GOOD or BAD, and automate interview scheduling, improving speed, consistency, and hiring quality.
